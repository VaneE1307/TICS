Definiciones de TIC

  • Hardware
  • Software
  • Internet

Hardware

Dispositivos de Entrada

Son aquellos que permiten al usuario introducir datos en la computadora, como el teclado, el ratón y el escáner.

Dispositivos de Salida

Permiten al usuario recibir información procesada por el computador, como el monitor, la impresora y los altavoces.

Dispositivos de Almacenamiento

Sirven para guardar información de manera temporal o permanente. Ejemplos: discos duros, memorias USB, CD.

Dispositivos de Procesamiento

Son los componentes que ejecutan las instrucciones y procesan los datos, como la CPU y la tarjeta gráfica.

Dispositivos de Comunicación

Permiten la transmisión de datos entre computadoras u otros dispositivos, como el módem, la tarjeta de red o el router.

Software

Es el conjunto de programas, instrucciones y reglas informáticas que permiten que el hardware funcione y que los usuarios realicen tareas. Se divide en software de sistema, de aplicación y de programación.

Internet

Es una red global que conecta millones de computadoras y dispositivos en todo el mundo, permitiendo el intercambio de información y servicios como el correo electrónico, redes sociales, páginas web, etc.
